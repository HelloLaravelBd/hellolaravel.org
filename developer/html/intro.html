HTML Story

What is HTML

HTML means HyperText Markup Language . It is not a programming language but it is the boss of web front-end . It describes and defines the content of a webpage . HTML builds a modern web site design with CSS & JS .

  • HyperText: It is text which connected with one or more another text or page .
  • Markup: HTML uses markup to annotate text, images, and other content for display in a Web browser. HTML markup includes special elements such as…
    <head>, <title>, <body>, <header>, <footer>, <article>, <section>, <p>, <div>, <span>, <img> and many others.

~ html code = open tag(<h1>) and close tag (</h1>) . ex: <h1>....</h1> . tag consist on angle <> .

Touch it

  • tag : <h1>
  • element: Open tag and close tag both that is element . <h1>....</h1>
  • attribute: Every element has atrribute as like href ,class , id , alt , poster etc.
